# 添加LCD图片显示功能

## 功能概述
在现有LCD驱动基础上添加图片显示功能，支持RGB565格式的图片数据。

## 实现方案

### 1. 创建图片数据结构体（lcd.h）
```c
typedef struct {
    uint16_t width;      // 图片宽度
    uint16_t height;     // 图片高度
    const uint16_t *data; // 图片数据指针（RGB565格式）
} lcd_image_t;
```

### 2. 添加图片显示函数（lcd.h和lcd.c）
```c
void lcd_show_image(uint16_t x, uint16_t y, const lcd_image_t *image);
```
功能：在指定位置显示图片

### 3. 提供示例图片数据（lcd.c）
创建一个简单的示例图片（如logo或图标），使用RGB565格式

### 4. 在main.c中添加使用示例
```c
lcd_show_image(10, 100, &example_image);
```

## 实现细节

### 图片显示函数实现
- 使用 `lcd_set_cursor` 设置光标位置
- 使用 `lcd_write_ram_prepare` 准备写入GRAM
- 遍历图片数据，逐像素写入LCD

### 图片数据格式
- 使用RGB565格式（16位颜色）
- 数据按行存储，每行从左到右

### 优势
- ✅ 简单易用，直接显示RGB565数据
- ✅ 高效，无需解码
- ✅ 灵活，支持任意尺寸图片
- ✅ 与现有LCD驱动完美集成

## 修改文件
- `Drivers/BSP/bsp_lcd/lcd.h`：添加结构体和函数声明
- `Drivers/BSP/bsp_lcd/lcd.c`：实现图片显示函数和示例数据
- `Core/Src/main.c`：添加使用示例
