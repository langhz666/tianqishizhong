# 简化汉字显示函数 - 统一字体大小参数

## 目标

将 `lcd_show_chinese()` 和 `lcd_show_chinese_32()` 合并为一个函数，通过参数指定字体大小。

## 修改步骤

### 1. 修改 lcd.h - 添加字体结构体和枚举

```c
// 字体大小枚举
typedef enum {
    FONT_SIZE_16 = 16,  // 16x16字体
    FONT_SIZE_24 = 24,  // 24x24字体（预留）
    FONT_SIZE_32 = 32,  // 32x32字体
} FONT_SIZE;

// 字体信息结构体
typedef struct {
    uint8_t size;           // 字体大小（16/24/32）
    const void *font_table;  // 字模表指针
} FONT_INFO;
```

### 2. 修改 lcd.h - 更新函数声明

```c
// 删除旧的函数声明
// void lcd_show_chinese_32(uint16_t x, uint16_t y, uint8_t *str, uint16_t color, uint16_t back_color);

// 修改为统一的函数声明
void lcd_show_chinese(uint16_t x, uint16_t y, uint8_t *str, uint16_t color, uint16_t back_color, FONT_SIZE font_size);
```

### 3. 修改 lcd.c - 添加字体信息表

```c
// 字体信息表
static const FONT_INFO font_info[] = {
    {16, cn_16x16},  // 16x16字体
    {32, cn_32x32},  // 32x32字体
};
```

### 4. 修改 lcd.c - 统一显示函数

```c
void lcd_show_chinese(uint16_t x, uint16_t y, uint8_t *str, uint16_t color, uint16_t back_color, FONT_SIZE font_size)
{
    // 根据字体大小选择对应的字模表和绘制函数
    // 支持自动换行
}
```

### 5. 删除 lcd.c 中的旧函数

* 删除 `lcd_draw_cn_32()` 函数

* 删除 `lcd_show_chinese_32()` 函数

### 6. 修改 main.c - 更新调用方式

```c
// 16x16字体
lcd_show_chinese(10, 60, (uint8_t*)"遇见司空", RED, WHITE, FONT_SIZE_16);

// 32x32字体
lcd_show_chinese(10, 140, (uint8_t*)"遇见司空", BLUE, WHITE, FONT_SIZE_32);
```

## 优势

* ✅ 只需一个函数，代码更简洁

* ✅ 通过参数指定字体大小，使用更灵活

* ✅ 易于扩展新的字体大小

* ✅ 减少代码重复

## 修改文件

* `Drivers/BSP/bsp_lcd/lcd.h`：添加结构体和枚举，更新函数声明

* `Drivers/BSP/bsp_lcd/lcd.c`：添加字体信息表，修改显示函数，删除旧函数

* `Core/Src/main.c`：更新调用方式
